Tool name: Browser
Purpose: Allows the user to browse websites and download images and media directly to "media pools".
Here, media pools are just save locations configured for all image tools, and video media pool is the save location for yt-dlp.

It runs inside the app itself with a modern webview.
It catches download requests and asks the user where they wanna save it to (not a file picker) with big vertically arranged buttons for different media pools for quick click.
